SmartCalc v1.0

Инструкция к использованию основного функционала калькулятора

Oсновное окно

На основном окне доступно поле ввода, куда можно ввести свое выражение при помощи клавиатуры или кнопок, расположенных в окне калькулятора(с клавиатуры необходимо вводить в окно ввода так же, как и написано на кнопках, например sin, cos).

При выборе унарных операторов(sin, cos, tan, acos и пр.) автоматически проставляется открывающая и закрывающая скобки, и курсор ставится внутри скобок, если необходимо посчитать выражение,к примеру sin(1) + sin(1). Пользователю нужно в ручную поставить курсор после скобки и написать свой знак.

Для получения результата выражения необходимо нажать кнопку =, к результату выражения можно в последующем так же применять любые операторы. Для получения ответа нажимается кнопка =.

Для очистки окна ввода/результата можно использовать клавишу Backspace, либо расположенную на интерфейсе калькулятора AC.

Так же есть возможность ввода символа X(можно и в нижнем регистре). В окне справа от ввода можно задать "Икс".

Oкно построения графика

После нажатия кнопки Plot открывается окно с графиком.

Пользователю доступны два окна ввода на этой "странице". X max и X min, после задания интервала от минимального, до максимального X можно построить график, нажав кнопку Create, если график необходимо построить исходя из результата функции, то в окно ввода необходимо ввести выражение, если в выражении присутствует X, то он туда подставляется из поля X min. Для того что бы построить новый график, можно ввести новые данные и заново нажать кнопку Create, либо нажать сперва Clear и следом Create, результат будет одинаковым.

Так же можно использовать скролл мыши для изменения масштаба графика. При зажатии ЛКМ (левой кнопки мыши) можно перемещаться по графику.

Кредитный калькулятор

Для перехода в режим Кредитного калькулятора в меню приложения и выбрать вкладку SmartCalc, следом нажать Calculate для перехода в режим расчета кредита.

Функционал кредитного калькулятора

После нажатия кнопки Calculate открывается новое окно в режиме "кредитный калькулятор".

Пользователю доступны ддля ввода несколько окон: "сумма кредита", "срок кредита" и "процентная ставка". После ввода значений в поля, необходимо выбрать тип платежа - Аннуитетный или Дифференцированный, следом нажать кнопку Рассчитать Откроется таблица с результатами, просмотреть все данные можно используя скролл мыши, либо скролл таблицы.
